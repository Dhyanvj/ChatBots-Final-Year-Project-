# Readme File

- Download the zip file provided which is 230 MB. It contains venv file and all the python code for all the four ChatBots.

# Please Follow the Steps Provide Here very carefully. Do not miss any steps.

1. Run this in the command in the terminal 'pip install -r requirements.txt'.
2. Run this in the command in the terminal 'python -m streamlit run app.py'.
3. Run this in the command in the terminal 'python -m streamlit run app1.py'.
4. Run this in the command in the terminal 'python -m streamlit run app2.py'.
5. Run this in the command in the terminal 'python -m streamlit run app3.py'.

6. Then Click on 'Run' then Click on 'Start Debugging' for the ChatWith.html.


If you Face some API Keys error. Then you can replace your own api in the file name .env:

1. OpenAI = https://platform.openai.com/account/api-keys
2. Deepgram = https://developers.deepgram.com/docs/authenticating
3. Elevnlabs = https://elevenlabs.io/docs/api-reference/text-to-speech
